# Fruit-Classification

# 🍎 Fruits Classification using Deep Learning (CNN)

## 📌 Project Overview

This project focuses on classifying different types of fruits using a Convolutional Neural Network (CNN). The model is trained on a dataset containing images of multiple fruit categories and learns to identify each fruit based on its visual features.

The project demonstrates the complete deep learning workflow, including data preprocessing, model building, training, evaluation, and prediction using TensorFlow and Keras.

---

## 🚀 Features

- Image preprocessing and normalization
- Multi-class fruit image classification
- CNN model built using TensorFlow and Keras
- Model training and validation
- Performance evaluation using accuracy and loss
- Predicts the class of unseen fruit images
