# Fruit-Classification

🍎 Fruit Classification using Deep Learning (CNN)

📌 Project Overview

This project uses Convolutional Neural Networks (CNN) to classify different types of fruits from images. The model learns visual features such as color, shape, and texture to accurately identify fruit categories.
the objective of this project is to demonstrate how Deep Learning can solve real-world image classification problems in agriculture, retail, food quality inspection, and smart inventory management.

🚀 Features

Multi-class fruit image classification

Image preprocessing and data augmentation

CNN model built using TensorFlow/Keras

Model training and evaluation

Prediction on new fruit images

Easy-to-understand implementation
